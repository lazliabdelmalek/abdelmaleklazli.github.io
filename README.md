# Portfolio — Abdelmalek Lazli

Portfolio professionnel présentant un double profil :
- **C#/.NET Full Stack Developer**
- **SAP S/4HANA Consultant (PS & PPM)**

Site statique développé en **HTML5, CSS3 et JavaScript Vanilla** (aucun framework), pensé pour être hébergé sur **GitHub Pages** et destiné à des recruteurs internationaux (Europe, Canada, Moyen-Orient).

## Structure du projet

```
portfolio/
│
├── index.html          → Page unique (structure sémantique de toutes les sections)
├── css/
│   └── style.css       → Feuille de style principale (design, animations, responsive)
├── js/
│   └── main.js          → Interactions (menu, scroll reveal, compteurs, formulaire, etc.)

├── assets/

├── documents/
│   └── CV_Abdelmalek_Lazli.pdf  → CV téléchargeable depuis le bouton "Download CV"
└── README.md
```

## Lancer le projet en local

Aucune installation n'est nécessaire, c'est du HTML/CSS/JS pur. Deux façons de tester :

1. **Le plus simple** : double-cliquer sur `index.html` → il s'ouvre dans votre navigateur.
2. **Recommandé** (évite certains problèmes de chargement de fichiers) : utiliser l'extension **Live Server** dans VS Code
   - Installer VS Code (https://code.visualstudio.com/)
   - Installer l'extension "Live Server" (icône Extensions dans la barre latérale → chercher "Live Server" → Install)
   - Clic droit sur `index.html` → "Open with Live Server"
